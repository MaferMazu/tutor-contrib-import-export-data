import-export-data plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/MaferMazu/tutor-contrib-import-export-data

Usage
-----

::

    tutor plugins enable import-export-data
